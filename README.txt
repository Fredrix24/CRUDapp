# CRUD Application (JavaFX + SQLite)

Это простое настольное приложение, разработанное с использованием JavaFX, которое демонстрирует базовые операции CRUD (Create, Read, Update, Delete) над сущностями, хранящимися в локальной базе данных SQLite.

## Технологии

*   **Язык:** Java
*   **GUI:** JavaFX
*   **База данных:** SQLite (JDBC)
*   **Доступ к данным (DAO):** Паттерн Data Access Object

## Особенности реализации

1.  **Структура:** Проект разделен на слои: Model, DAO, Service и View.
2.  **Инициализация БД:** База данных (файл `entities.db`) создается автоматически при первом запуске. Таблица `entities` создается с помощью встроенного SQL в конструкторе `EntityDAOImpl`.
3.  **Пагинация:** Метод `readAll` в DAO поддерживает пагинацию для эффективной загрузки данных.
4.  **Поиск (Фильтрация):** В интерфейсе пользователя (MainView) реализован поиск в реальном времени по полям "Name" и "Description" с использованием `FilteredList` JavaFX.

## Структура Проекта

*   `com.example.model`: Класс `Entity`.
*   `com.example.dao`: Интерфейс `EntityDAO` и его реализация `EntityDAOImpl` (работа с SQLite).
*   `com.example.service`: Бизнес-логика (`EntityService`).
*   `com.example.`: Основной класс приложения (`MainView`).

## Как запустить локально (Предполагая Maven/Gradle)

1.  Убедитесь, что у вас установлены **JDK** (Java Development Kit) и **JavaFX SDK**.
2.  Если используется сборщик (Maven/Gradle), запустите приложение с помощью соответствующей цели (например, `mvn javafx:run` или `gradle run`).
3.  При первом запуске будет создан файл `entities.db` в рабочей директории.
